Documentation Page for MyFlix

This api will allow you to receive information on movies, directors, and genres as well as create your own profile so data can be saved about your favorite movies

Business Logic URL HTTP Method Query Parameter Request data format Response data body format
Get a list of all movies /movies GET None None A JSON object holding data about all movies
Get data about a single movie by name /movies/:Title GET :Title None A JSON object holding data about a single movie, containing a description, genre, director, image URL, and whether it's featured or not
Get data about a genre (description) by name/title /movies/genre/:Name GET :Name None A JSON object holding data about a specific genre
Get data about a director by name /movies/director/:Name GET :Name None A JSON object holding data about a director (bio, birth year, death year)
Allow new users to register /users POST None A JSON object holding data about the user to add like: { "Username": "", "Password": "", "Email: " "", "Birthday": ""} A JSON object holding data about the user that was added, including an ID
Allow users to update their user info /users/:Username PUT :Username None A text message indicating the username was updated
Allow users to add a movie to their list of favorites /users/:Username/movies/:_id POST :Username :_id None A text message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE :Username :id None A text message indicating the movie was removed
Allow existing users to deregister /users/:Username DELETE :Username None A text message indicating the user was removed
